# рҹ“– Overview

Touchless Gesture-Controlled OS Interface is a **Computer Vision** project developed using **Python, OpenCV, MediaPipe, and PyAutoGUI**. The system captures live video from a webcam, detects hand landmarks, recognizes gestures, and converts them into operating system actions such as cursor movement, mouse clicks, scrolling, drag-and-drop, screenshots, and zoom.

No external hardware is requiredвҖ”just a standard webcam.
